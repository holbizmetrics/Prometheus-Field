### Strategies

When this capability resonates, it contributes:

1. **Generate before filtering.** High T, low Λ. Let noise produce candidates. Stochastic resonance needs the noise.
2. **Collide deliberately.** Take two unrelated fuel sources, smash them together. The barrier height determines optimal noise level.
3. **Transform systematically.** Apply operators to candidates. Single pass for quick work, chained for depth.
4. **Crystallize by convergence.** If 3+ independent paths find the same structure, it's real. If only one path finds it, it might be noise.
5. **Capture before cooling.** Write insights down while T is still warm. Cold notes lose the connections.

## Flow Patterns

Creative synthesis doesn't follow a fixed pipeline. But it has common flow patterns:

**Full synthesis** (was FORGE default):
```
FUEL → high-D interference → crystallize what survives → capture
Field: V low → D rises → candidates form → V rises as structure emerges → capture
```

**Collision** (was FORGE COLLIDE):
```
Two sources → maximum D → deliberate interference → capture matches
Field: D spikes → T rises from collision energy → crystallize or scatter
```

**Fountain** (was FORGE FOUNTAIN):
```
Finding → suggests question → question → finding → ...
Field: moderate V (sustained direction) + moderate T (warm, not hot) + rising D
Exit: two consecutive rounds produce nothing new (T dropping naturally)
```

**Reframe** (was FORGE REFRAME):
```
Stuck point → invert → see what changes
Field: V drops (release attachment) → apply one operator → V rises if insight forms
```

These aren't prescribed stages. They're what high-D, low-Λ, low-V thinking naturally does.
